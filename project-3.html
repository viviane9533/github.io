Company - UC Riverisde Computer Science Class

Role - UI/UX Designer and Frontend Developer

Focus - Interface Design

In my Introduction to Software Engineering class we created a project. This is a team project (5 students per team). The goal is to build a data analytics app mostly from scratch (i.e., without using any database or analytics library). The app provides predefined data analytics for gaining insights from the data. Initially, the app can be populated with some public dataset in CSV format.

Task

We need a data analytics application to present the NBA Dataset. Also use data analytics to show visual representation of the data.

Design

I was responsible in creating each Interface for the project. We wanted a user-friendly and appealing interface which is easy to navigate. I start off creating wireframes on Miro.


The Dashboard was the main page users would see as it is the homepage that links to all features of the app. Since a lot of users where going to see this portion of the application, I wanted to make sure it looked good and was functing well.


I decided that each feature would have a different color scheme. For example the To-Do list was blue colored and the Calender was orange colored. Here is an example of the Calender.


The Login Page was the main part of the project I was responsible for. I used HTML/CSS to create all the features and buttons. One part of the project that many liked was the background of the login page. Since the whole program was different colors I decided to have an interactive gradient moving background for the login page. As you can see in the video, the login page's background changes colors using all the colors.

Challenges

Our group had a hard time connecting a database so that user's information would be saved while using the app. This was solved by getting help from volunteering Google Engineers.

Accomplishments

This program was one of the first times I really had a hands-on experience with coding. For this reason I felt proud to have coded the login page as I never had an experience doing so. Our project functioned very well once it was time to present and others were impressed by the different features our app can do.

What's Next For Lifestyle

We hope to add connections to emails and external services to make students lives easier. One feature I want to create is a language feature so that our application can be used by students and people all over the world.

Overview

Google's CSSI program was one of the first times I really coded. It was an amazing experience and I founded my passion for Computer Science and specifically designing through this program.
